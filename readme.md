# Set up

Make sure that you have a [react native environment](https://reactnative.dev/docs/environment-setup) set up

Clone repo

```sh
    git clone https://github.com/uwemneku/hims-clone.git
```

install dependencies

```sh
    yarn
```

start the expo app

```sh
    npx expo start
```
